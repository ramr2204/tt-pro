ESTAMPILLAS PRO

MANUAL DEL ADMINISTRADOR DEL APLICATIVO EN LÍNEA

En este manual, se describe el funcionamiento del aplicativo de estampillas. Como primer paso, se debe abrir un navegador de internet y digitar la dirección: www.softwareenlanube.net:8086/estampillas-pro/, allí debe ingresar el usuario y contraseña (el cual ha sido proporcionado por el administr ador del sistema) y dar clic en ingresar.

Una vez ingresado al sistema, se direcciona a la pagina principal donde encontrará los contratos que se encuentran en proceso de liquidación.

PROCESO CONTRATACIÓN

Contiene las opciones habilitadas para la administración del proceso de liquidación de contratos.

  • PARAMETROS

    • Tipos de Régimen Agrega, lista y actualiza los tipos de régimen para el contratista.

      • Lista de Tipos de Régimen Muestra el listado de los regimenes creados para los contratistas y la opción de actualizarlos

      • Agregar de Tipos de Régimen Muestra el formulario para la creación de un nuevo tipo de régimen.

    • Tipos de Contrato Agrega, lista y actualiza los tipos de contrato.

      • Listar Tipos de Contrato Muestra el listado de los tipos de contrato creados y la opción de actualizarlos.

      • Nuevo Tipo de Contrato Muestra el formulario para la creación de un nuevo tipo de contrato.

  • CONTRATOS

    • Contratistas Agrega, lista y actualiza los contratistas.

      • Listar Contratistas Muestra el listado de contratistas creados y la opción de actualizarlos.

      • Nuevo Contratista Muestra el formulario para la creación de un nuevo contratista.

    • Contratos Agrega, lista y actualiza los contratos.

      • Listar Contratos Muestra el listado de los contratos creados y la opción de actualizarlos.

      • Nuevo Contrato Muestra el formulario para la creación de un nuevo contrato.

      • Importar Contratos Contiene un formulario para la importación por web- service al Sistema de Contratación.

PROCESO ESTAMPILLAS

Contiene las opciones habilitadas para la administración de los parametros y asignación de papeleria.

  • ESTAMPILLAS

    • Estampillas Pro Agrega, lista y actualiza los tipos de Estampillas Pro.

      • Lista de Estampillas Pro Muestra el listado de los tipos de estampillas creadas y la opción para actualizarlas.

      • Agregar Estampillas Pro Muestra el formulario para la creación de un nuevo tipo de estampilla.

    • Inventario Agrega, lista y actualiza el inventario de papeleria "rotulos", utilizados para la impresion de las estampillas pro .

      • Lista de Impresiones Muestra el listado de las estampillas impresas y su estado.

      • Anular Papel Muesta el formulario para anular papel, donde se ha impreso la estampilla.

      • Papeleria Ingresada Muestra el listado de liquidador el cual tiene papeleria asignada.

      • Ingresar Papeleria Muesta el formulario para asignar papeleria a cada liquidador.

      • Reasignar Papeleria Muesta el formulario para re- asignar papeleria de un liquidador a otro.

    • Trámites Agrega, lista y actualiza los tipos de tramites disponibles según la resolución actual.

      • Agregar Trámite Muestra el formulario para la creación de un nuevo tipo de tramite.

      • Lista de Trámites Muestra el listado de los tipos de tramites.

    • Pagos Muestra el formulario mediante el cual se habilita la carga de un archivo plano suministrado por el banco, que contiene la relación de los pagos por concepto de estampillas, realizados durante el día.

      • Cargar Archivos de Pagos

PROCESO LIQUIDADOR

Contiene las opciones habilitadas para realizar el proceso de liquidación de contratos y/o tramites.

  • Liquidaciones

    • Liquidar

      • Contratos Muestra el listado de contratos sin liquidar, liquidado, legalizado.

        Para visualizar la información de la liquidación dar click en los botones.

        Sin liquidar

        Si el contrato no se encuentra liquidado,el valor de las estampillas es editable y visible el boton liquidar.

        Liquidado
        identificado con el botón.

        Muestra el formulario para el cargue del soporte de la consignacion en formato pdf, jpeg, gif, png, tif. obligatorio para cada uno de las estampillas liquidadas, igualmente la fecha que tiene el comprobante, la cual (No debe ser mayor a la fecha actual).

        Legalizado
        identificado con el botón.

        Muestra el formulario para impresion de la estampilla. Para el perfil administrador se encuentra inhabilidato el botón de imprimir ya que esta función es propia del perfil liquidador.

    • Tramites Muestra el listado de tramites sin liquidar, liquidado, legalizado. Nota: La liquidación y legalizacion del tramite se debe realizar el mismo proceso que se realiza para los contratos

    • Consultar Muestra el listado de tramites y contratos realizados, permite realizar el filtro del informe para una fecha en especifico.

  • Menús Agrega, lista y actualiza el menú del aplicativo.
